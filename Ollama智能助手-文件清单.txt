═══════════════════════════════════════════════════════════════
  快速命令参考
═══════════════════════════════════════════════════════════════

安装Ollama：
  Windows:   winget install Ollama.Ollama
  Linux:     curl -fsSL https://ollama.ai/install.sh | sh
  macOS:     brew install ollama

启动服务：
  ollama serve

下载模型：
  ollama pull llama2        # 通用对话，7GB
  ollama pull mistral       # 轻量快速，4GB
  ollama pull qwen          # 中文优化，7GB

查看模型：
  ollama list

测试对话：
  ollama run llama2

检查API：
  curl http://localhost:11434/api/tags

启动平台：
  python app.py

# Ollama智能助手 - 使用说明

## ✅ 已完成配置

你的RL4CO平台已成功集成**嵌入式AI智能助手**！

对话界面已直接显示在首页右侧，随时可用。

---

## 🚀 快速开始（3步）

### 第一步：启动Ollama服务

```bash
ollama serve
```

### 第二步：下载模型

```bash
# 推荐：通用对话模型
ollama pull llama2

# 或者：其他模型
ollama pull mistral    # 轻量快速
ollama pull qwen       # 中文优化
ollama pull codellama  # 代码专用
```

### 第三步：使用助手

1. 启动RL4CO平台：`python app.py`
2. 打开浏览器访问首页
3. 右侧即可看到AI智能助手面板
4. 选择模型，开始对话！

## ⚠️ 常见问题

### Q: 显示"Ollama未运行"？
```bash
ollama serve
```

### Q: 没有可用模型？
```bash
ollama pull llama2
```

### Q: AI回复太慢？
- 切换到更小的模型（如mistral）
- 关闭其他占用资源的程序

### Q: 历史记录丢失？
- 避免清理浏览器缓存
- 不使用无痕模式

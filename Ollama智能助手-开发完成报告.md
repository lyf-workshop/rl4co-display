## 🚀 部署步骤

### 第一步：验证文件完整性

```bash
# 检查文件是否存在
ls static/css/ollama-chat.css
ls static/js/ollama-chat.js
ls docs/Ollama智能助手*.md
```

### 第二步：安装Ollama

**Windows系统**：
```powershell
# 下载安装程序
# https://ollama.ai/download/windows

# 或使用winget
winget install Ollama.Ollama
```

**Linux系统**：
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

**macOS系统**：
```bash
brew install ollama
```

### 第三步：启动Ollama服务

```bash
# 启动服务
ollama serve

# 验证服务
curl http://localhost:11434/api/tags
```

### 第四步：下载模型

```bash
# 推荐模型（任选其一）
ollama pull llama2        # 通用对话，7GB
ollama pull mistral       # 轻量快速，4GB
ollama pull qwen          # 中文优化，7GB

# 验证模型
ollama list
```

### 第五步：启动RL4CO平台

```bash
# 激活虚拟环境
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 启动Flask服务
python app.py
```

### 第六步：访问测试

1. 打开浏览器访问：`http://localhost:5000`
2. 在首页右下角应该看到机器人图标🤖
3. 点击图标打开对话面板
4. 选择模型并测试对话

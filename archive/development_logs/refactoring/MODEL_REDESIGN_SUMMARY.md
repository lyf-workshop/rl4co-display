## 🛠️ 维护指南

### 添加新模型
1. 在 `model_database.py` 中添加模型字典
2. 确保包含以下字段：
   ```python
   "ModelName": {
       "name": "缩写名",
       "full_name": "完整名称",
       "category": "Constructive (AR)",  # 必须
       "category_zh": "构造方法（自回归）",  # 必须
       "year": "2024",
       "paper_url": "https://...",
       "paper_venue": "会议名称",
       # ... 其他字段
   }
   ```
3. 页面会自动更新，无需修改HTML

### 修改分类
1. 确保新分类在两个视图中都有对应
2. 更新 `category_info` 字典
3. 添加合适的图标和描述

### 调整样式
- 主题色：修改 `#667eea` 和 `#764ba2`
- 卡片大小：修改 `minmax(280px, 1fr)`
- 动画速度：修改 `transition` 和 `animation`

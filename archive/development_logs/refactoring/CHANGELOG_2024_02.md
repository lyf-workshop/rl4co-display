## 🔧 修复的问题

### Bug #1：FFSP 环境名称不匹配 ✅ FIXED
**问题：**
```
❌ Unknown environment name 'ffsp'
```

**原因：**
- FFSPEnv.name = "FFSP"
- AttentionModelPolicy 没有 'ffsp' embedding
- 但有 'fjsp' embedding

**修复：**
```python
# base_trainer.py - 添加环境名称映射
env_name_mapping = {'FFSP': 'fjsp'}
```

**状态：** ✅ 已修复，FFSP 可正常使用

---

### Bug #2：FJSP/FFSP 前端 ID 冲突 ✅ FIXED
**问题：**
- FJSP 和 FFSP 使用相同的字段 ID
- 导致参数收集错误

**修复：**
```html
<!-- FJSP -->
<input id="fjsp-num-jobs" ...>    ✅ 唯一

<!-- FFSP -->
<input id="ffsp-num-stage" ...>   ✅ 唯一
```

**状态：** ✅ 已修复，参数收集正常

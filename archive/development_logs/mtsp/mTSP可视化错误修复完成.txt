错误1：AttributeError - 'plots_dir' 不存在

根本原因：
  属性名称错误，使用了不存在的 self.plots_dir

详细分析：
  1. BaseTrainer 初始化时设置的是 self.user_plots_dir
     位置: modules/rl_training/base_trainer.py 第323行
     
     代码:
     self.user_plots_dir = get_user_plot_dir(user_id)
  
  2. TSPTrainer 正确使用了 self.user_plots_dir
     位置: modules/rl_training/tsp_trainer.py 第125行
     
     代码:
     plot_path = os.path.join(self.user_plots_dir, plot_filename)
  
  3. MTSPTrainer 错误使用了 self.plots_dir
     位置: modules/rl_training/mtsp_trainer.py 第98行
     
     错误代码:
     anim_path = os.path.join(self.plots_dir, anim_filename)  # ❌
     
  4. 应该使用 self.user_plots_dir
     正确代码:
     anim_path = os.path.join(self.user_plots_dir, anim_filename)  # ✅

  def generate_visualizations(self, env, model, trainer, checkpoint_path):
      try:
          # 1. 生成可视化数据
          ...
          
          # 2. 保存文件（使用正确的目录）
          for i in range(num_instances):
              filename = f'viz_{i}.png'
              file_path = os.path.join(self.user_plots_dir, filename)  # ✅
              
              # 生成文件
              create_visualization(data, file_path)
              
              # 保存到数据库
              if self.bg_file_manager:
                  self.bg_file_manager.save_file_record(
                      user_id=self.user_id,
                      session_id=self.session_id,
                      filename=filename,
                      file_type='plot',
                      file_path=file_path
                  )

              paths.append(f'/static/model_plots/user_{self.user_id}/{filename}')  # ✅

          trainer.save_checkpoint(checkpoint_path)

                  file_path=checkpoint_path
              )

              'training_curve': self.training_status[self.session_id].get('plot_url', ''),

          # 返回空字典（不是空列表）

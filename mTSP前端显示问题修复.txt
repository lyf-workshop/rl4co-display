    后端返回的字段名与前端期望的字段名不一致

    - 功能看起来没有工作
    - 但实际上后端已经正确执行
  
  解决：
    1. 查看其他训练器（如 TSPTrainer）的返回格式
    2. 查看前端代码期望的字段名
    3. 确保返回字段名与前端期望一致
  
  正确的返回格式（所有训练器必须一致）：
  
    return {
        'plot_paths': [...],        # ✅ 静态对比图路径列表

        'checkpoint_path': '...'    # ✅ 检查点路径
    }
  
  错误示例：
  
    return {
        'comparison_paths': [...],  # ❌ 错误！前端不认识这个字段

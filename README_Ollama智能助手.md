# 🤖 Ollama智能助手 - 快速说明

## 📌 这是什么？

Ollama智能助手是集成在RL4CO平台首页的AI对话模块，可以与本地运行的Ollama大模型进行实时对话。

**核心特性**：
- 🎯 悬浮式设计，不遮挡主界面
- 💬 流式对话，实时响应
- 🔄 多模型切换
- 💾 历史记录保存
- 📱 响应式设计

---

## ⚡ 快速开始（3步）

### 1️⃣ 安装Ollama

```bash
# Windows
winget install Ollama.Ollama

# Linux
curl -fsSL https://ollama.ai/install.sh | sh

# macOS
brew install ollama
```

### 2️⃣ 下载模型

```bash
ollama serve
ollama pull llama2
```

### 3️⃣ 使用助手

1. 打开RL4CO首页
2. 点击右下角🤖
3. 选择模型开始对话

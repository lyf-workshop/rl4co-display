## 🎭 动画效果列表

### 关键帧动画
1. **fadeInUp** - 页面整体淡入上浮（0.6s）
2. **shimmer** - 顶部装饰条流动闪烁（3s 循环）
3. **floatGlow** - 头部背景光晕浮动（10s 循环）
4. **titleGradient** - 标题渐变滚动（5s 循环）
5. **slideIn** - 内容区域滑入（0.5s）
6. **ripple** - 徽章波纹效果（0.6s）

### 过渡动画
- 所有交互元素使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动函数
- 悬浮效果过渡时间：0.3s - 0.4s
- Transform 动画优先，保证性能

# Ollama智能助手 - 快速开始 ⚡

## 🎯 三步开始使用

### 第一步：安装Ollama

**Windows**
```powershell
# 访问 https://ollama.ai/ 下载安装程序
# 或使用winget
winget install Ollama.Ollama
```

**Linux/Mac**
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

### 第二步：下载模型

```bash
# 启动服务
ollama serve

# 下载推荐模型（任选一个）
ollama pull llama2        # 通用对话（7GB）
ollama pull mistral       # 轻量快速（4GB）
ollama pull qwen          # 中文优化（7GB）
```

### 第三步：使用助手

1. 打开RL4CO平台首页
2. 点击右下角🤖图标
3. 选择模型并开始对话！

## ⚠️ 常见问题

**Q: 显示"Ollama未运行"？**
```bash
# 启动Ollama服务
ollama serve
```

**Q: 没有可用模型？**
```bash
# 下载模型
ollama pull llama2
```

**Q: AI回复太慢？**
- 切换到更小的模型（mistral）
- 关闭其他程序释放内存
- 考虑使用GPU加速

**Q: 历史记录丢失？**
- 避免清理浏览器缓存
- 不使用无痕模式

## 💻 系统要求

**最低配置**
- CPU: 4核心
- 内存: 8GB RAM
- 磁盘: 10GB可用空间

**推荐配置**
- CPU: 8核心
- 内存: 16GB RAM
- GPU: NVIDIA (可选，加速推理)
- 磁盘: 20GB SSD
